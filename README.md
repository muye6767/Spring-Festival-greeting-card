## 💻 技术实现参考

### 特效说明

| 特效名称 | 技术要点 | 应用场景 |
|----------|----------|----------|
| 打字机效果 | 逐字显示、光标闪烁 | 开场白、祝福语 |
| 雪花飘落 | CSS动画/Canvas | 背景氛围 |
| 福字爆炸 | 粒子效果、缩放动画 | 点击交互、高潮时刻 |
| 淡入淡出 | opacity过渡 | 页面切换、元素显现 |
| 心跳效果 | scale缩放动画 | 强调重点、情感表达 |

### 推荐技术栈
- **基础**: HTML5 + CSS3 + JavaScript
- **动画**: CSS Animations / GSAP / Anime.js
- **粒子效果**: Canvas API / particles.js
- **响应式**: Flexbox / Grid + 媒体查询

---

## 🎯 使用建议

1. **文件命名规范**: 使用英文或拼音命名，避免中文路径在某些服务器上显示异常
2. **图片优化**: 将图片放入 `img/` 目录，建议使用 WebP 格式压缩
3. **多语言支持**: 为贺卡添加语言切换功能（中/英/拼音）
4. **移动端适配**: 确保在各种设备上都能正常显示
5. **性能优化**: 动画使用 `transform` 和 `opacity` 以启用 GPU 加速

---

## 📌 快速开始

1. 打开 `基础版春节贺卡.html` 查看基础结构
2. 根据需求选择特效版本（打字机/雪花/心跳等）
3. 替换 `img/` 目录中的图片资源
4. 修改文案内容（参考上方祝福语）
5. 测试所有动画效果
6. 部署到服务器或打包分享
